Preparatory evidence · one-minute class

How continuity was tested

One deterministic task fixture was run through a chat-only baseline and a Kungfu continuation path. Both paths used the same task and exact oracle.

Baseline

The fresh worker received no continuation input. It could not recover the required next action, so the verdict was unsupported.

Kungfu path

The fresh worker received one durable fact, recovered the required next action, and matched the fixture's exact continuation oracle.

What was retained

The public projection, raw evidence index, path reports, and animation production pack share one run identity and content roots.

Versions and method

The retained report binds its source commit, fixture version, contract version, run identity, redaction rules, and static fallback.

Limitations

This is a deterministic fixture-worker smoke test. It is not a native or hosted model-agent comparison, a multi-hour durability result, a user-retention study, or proof that Kungfu outperforms Codex, Claude, or another agent.
