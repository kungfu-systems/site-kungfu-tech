Long-running agent work control pane

See when your agents are making progress, getting stuck, or wasting tokens.

Kungfu gives long-running agent work a local control pane: cost, progress, evidence, and next actions.

Use Codex, Claude, Copilot, or your own execution surface. Kungfu is not another agent; it is the local-first place to verify whether delegated work is actually moving, blocked, ready, or done.

Open source core, commercial stewardship.

Kungfu is commercially operated by Kungfu Origin Technology Limited.

Kungfu core software is released under the Apache License 2.0. The business model is not based on private-code scarcity.

The local core remains Agent-native and journal-first, so humans and coding agents can inspect the same evidence instead of trusting an opaque dashboard.

Future commercial services may provide optional cloud, team, compute, storage, support, or governance capabilities without making the local core depend on login or hosted infrastructure.

Public feedback, bug reports, feature requests, and code contributions start on GitHub. Security issues should use GitHub private vulnerability reporting.
