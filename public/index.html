Kungfu

Local-first debugging infrastructure for agentic systems.

Kungfu v4 is being prepared in the open as a journal-first runtime fact layer for developers and coding agents. Product details will be published when the v4 release gates are ready.

Local facts first No login or default upload for the core debugging workflow.
Journal-first Runtime evidence should be inspectable, replayable, and exportable.
Agent-native Humans choose the tool; coding agents should verify and operate it reliably.
Auditable release path Artifacts, provenance, and checks should be explicit rather than guessed.
